# Support

[English](#english) | [简体中文](#简体中文)

## English

- Use [Discussions](https://github.com/patton174/coco-framework/discussions)
  for usage questions, integration design, and ideas.
- Use the Bug Report form for reproducible framework defects.
- Use the Feature Request form for a concrete reusable framework capability.
- Use [private vulnerability reporting](https://github.com/patton174/coco-framework/security/advisories/new)
  for security issues.

This open-source repository does not provide a private support SLA. Include the
Coco version, JDK, Spring Boot version, affected module, minimal reproduction,
and relevant logs so the community can act on the report.

## 简体中文

- 使用 [Discussions](https://github.com/patton174/coco-framework/discussions)
  咨询接入、方案设计和想法。
- 可复现的框架缺陷使用 Bug Report 表单。
- 可复用的框架能力建议使用 Feature Request 表单。
- 安全问题使用
  [私密漏洞报告](https://github.com/patton174/coco-framework/security/advisories/new)。

开源仓库不提供私有支持 SLA。提交问题时请提供 Coco 版本、JDK、Spring Boot 版本、
影响模块、最小复现和必要日志。
